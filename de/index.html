Verschlüsselung auf dem Gerät

Jedes Foto, Video und Album wird mit AES-256 auf deinem iPhone verschlüsselt. Der Schlüssel verlässt dein Gerät nie.

Face ID und Code

Entsperre mit Face ID oder einem privaten Code, getrennt von dem deines Telefons. Es sperrt, sobald du es verlässt.

Schein-Tresor

Ein zweiter Code öffnet einen glaubwürdigen Schein-Tresor, damit ein unter Druck verratener Code nie den echten preisgibt.

Verschlüsseltes iCloud-Backup

Sichere optional in deinem eigenen iCloud, verschlüsselt, bevor es dein Gerät verlässt. Selbst wir können es nicht lesen.

Vor Screenshots geschützt

Dein Tresor ist vor Screenshots, Bildschirmaufnahme und AirPlay geschützt, damit nichts versehentlich nach außen gelangt.

Tarnkamera

Nimm Fotos und Videos direkt im Tresor auf, sodass sie nie in deinen normalen Aufnahmen landen.
